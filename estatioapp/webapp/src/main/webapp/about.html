Estatio

This is a prototype of the Estatio application, implemented using Apache Isis.

The app can be accessed here, using, the preconfigured username to login with is: sven, password: pass.

The back-end database runs in-memory, which means that any changes you make will not be persisted (they are lost once the server is restarted).
